## 🚀 Features

- Upload image (Gallery + Camera)
- Multi-style generation (Cartoon, Anime, Flat, Sketch)
- Progressive loading UI (skeleton + async rendering)
- Before/After comparison slider
- Download generated images
- Share via native share sheet
- Retry per style
- Style selection
- Smooth animations

---

## 🧠 Tech Stack

- React Native (Expo)
- TypeScript
- Expo Image Picker
- Expo Sharing / FileSystem
- Custom UI components
